# StickerAgent - 스티커 프린터 에이전트

## 역할
주문 리스트에서 원단 스티커 + 배송 스티커를 자동 생성하는 에이전트.
프린터: Xprinter XP-DT108B LABEL (100x100mm)

## 스티커 종류

### 1. 원단 스티커 (물건에 붙이는 용)
- 생성 조건: 원단명 + 수량이 있으면 생성
- 표시: 원단명 / #색상번호 수량Y
- 일신→한주 가공건: 원단명 / #색상번호 수량Y / 가공종류(빨간색)

### 2. 배송 스티커 (택배에 붙이는 용)
- 생성 조건: 운송이 "대신택배" 또는 "퀵" + 하위업체 없음 + 주소 있음
- 표시: 운송+결제 / 주소 / 수신자 / 전화번호 / 발신(있으면)

## 입력 데이터 컬럼 (탭 구분)

```
cols[0] = 주문업체
cols[1] = 원단명
cols[2] = 색상번호
cols[3] = 수량
cols[4] = 하위업체 (페르/한주/대성 등 - 있으면 배송스티커 제외)
cols[5] = 운송 (대신택배/퀵/배달/일신/로젠)
cols[6] = 결제 (현불/착불 - 안적혀있으면 착불)
cols[7] = 주소
cols[8] = 수신자
cols[9] = 전화번호
cols[10] = 비고
```

## 필터링 규칙

### 배송 스티커 생성
- 대신택배: 하위업체(cols[4]) 비어있을 때만
- 퀵: 하위업체 비어있을 때만
- 하위업체에 페르/한주/대성/현대 등 있으면 → 배송 스티커 생성 안 함

### 일신 → 한주섬유 (가공건)
- 운송 = "일신" + 수신자에 "한주" 포함
- 배송 스티커 불필요
- 원단 스티커에 가공종류 추가 (비고의 / 앞부분)
- 예: "발수 / 크레아" → "발수가공" 빨간색 표시

### 따옴표(") 처리
- " = 위 행과 동일한 값
